Projects


Jndi Evaluator

Jndi Evaluator is a tool that can be used to validate JNDIs. Some problems that previously occurred included uncertainty over whether configuration occurs twice due to having to eyeball files and uncertainty after a restart if the proper JNDI was being picked up. The JNDI Evaluator solves these problems by 1. Checking duplicate entries in JNDI 2. Reporting differences between JNDI configuration files 3. Providing security analysis in JNDI configuration – password change. You can view the project on GitHub.


Cadenza

Cadenza is an application where people can rate their favorite songs, search for specific music, or discover new tunes. The SQL database I created stores information about various artists, songs, albulms, recording labels, the user, etc. A web front end was created using PHP. On the website the user can search the database by various categories (i.e artist, song name, album, etc.), modify their rating of various songs, and receive recommendations for music. You can view the project on GitHub.
